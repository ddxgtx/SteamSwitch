# Steam Switch 2.0

2.0 是一次界面和快捷入口的大版本更新，重点加入桌面悬浮窗、任务栏常驻增强、游戏账号绑定和更完整的功能开关。

## 新增

- 桌面悬浮窗：支持固定账号切换、固定游戏启动、拖动位置保存、始终置顶、锁定位置和透明度调节。
- 任务栏常驻增强：右侧/自动模式改为右边框锚定，新增固定游戏后向左扩展，避免挤占 Windows 托盘区域。
- 游戏界面增强：支持本地游戏扫描、游戏图标读取、游戏账号绑定、固定游戏快捷启动。
- 设置项扩展：新增桌面悬浮窗、启动前确认、启动时自动扫描游戏、Steam 库界面注入等功能开关。
- 液态玻璃视觉：主界面、设置页、任务栏常驻和桌面悬浮窗统一为更现代的浅色玻璃风格。

## 修复

- 修复任务栏常驻新增游戏后右对齐失效的问题。
- 修复任务栏常驻遮挡 Windows 隐藏图标按钮的问题。
- 修复高 DPI 缩放下任务栏定位偏差。
- 修复固定账号变化后任务栏常驻未及时刷新的问题。
- 改进游戏图标读取和无图标时的回退显示。

## 风险提示

Steam 库界面注入功能会开启 Steam CEF 调试端口，并向 Steam 客户端页面执行脚本。该功能可能带来 Steam 客户端稳定性、平台规则或反作弊相关风险。建议默认关闭，仅在明确需要时临时开启；不建议在 VAC、竞技或反作弊敏感游戏运行期间使用。
