Steam Switch

一站式 Steam 多账号管理工具

账号切换

自动读取 Steam 登录账号，一键切换并启动 Steam，支持系统托盘快速切换

游戏绑定

扫描本机游戏，为游戏绑定默认账号，固定到快捷入口一键启动

任务栏常驻

将账号和游戏嵌入 Windows 任务栏，支持液态玻璃效果和圆角模式

桌面悬浮窗

可拖动的快捷入口，支持 6 种玻璃颜色、透明度调节、始终置顶

Steam 库注入

在游戏详情页注入「切换启动」按钮，直接使用绑定账号启动游戏

自动更新

启动时自动检查 GitHub 最新版本，发现新版本支持一键更新
