功能特性

为 Steam 多账号用户打造的高效管理工具

🎮

账号切换

自动读取本机 Steam 账号，一键切换并启动，支持系统托盘快速操作，切换时气泡提醒

🔗

游戏绑定

扫描本机已安装游戏，为每个游戏绑定默认启动账号，固定到快捷入口一键启动

📌

任务栏常驻

将账号和游戏嵌入 Windows 任务栏，支持液态玻璃效果、圆角模式、位置调节

🪟

桌面悬浮窗

可拖动的快捷入口，支持 6 种玻璃颜色、透明度调节、始终置顶，不出现在 Alt+Tab

💉

Steam 库注入

在游戏详情页注入「切换启动」按钮，点击直接使用绑定账号启动游戏

🔄

自动更新

启动时自动检查 GitHub 最新版本，发现新版本弹窗提示，支持一键下载安装

🎨

主题切换

暗黑模式与白色模式一键切换，所有界面元素跟随主题自动变化

⚙️

独立设置

任务栏与悬浮窗完全独立配置，窗口大小、图标大小、效果均可分别调节

🛡️

安全稳定

不修改 Steam 文件，不注入游戏进程，仅读取本地配置，开源可审计

界面预览

现代化设计，暗黑/白色双主题
